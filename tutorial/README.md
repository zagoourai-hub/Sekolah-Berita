# Tutorial Docker Project Sekolah Berita

Folder ini berisi panduan sederhana untuk menjalankan project dengan Docker di Windows.

Urutan baca yang disarankan:

1. `docker-installation.md`
   File ini menjelaskan cara install Docker Desktop sampai siap dipakai.
2. `docker-structure.md`
   File ini menjelaskan isi Docker project ini dengan bahasa yang lebih ringan.

Jika tujuan Anda hanya ingin project cepat jalan, fokus utama ada di `docker-installation.md`.
